# Website File Structure

## Root Directory
- index.html (Homepage)
- about.html (About page)
- people.html (People page)
- publications.html (Publications page)
- research.html (Research page)
- news.html (News page)
- gallery.html (Gallery page)
- contact.html (Contact page)

## CSS Directory (/css)
- style.css (Main stylesheet)
- responsive.css (Responsive design rules)

## JavaScript Directory (/js)
- main.js (Main JavaScript file)

## Images Directory (/images)
- logo.png (Lab logo)
- faculty/ (Directory for faculty photos)
- students/ (Directory for student photos)
- gallery/ (Directory for gallery images)
- research/ (Directory for research-related images)

## Assets Directory (/assets)
- fonts/ (Custom fonts if needed)
- icons/ (Icons for the website)
- documents/ (PDFs and other documents)
